# Message Refiner Chrome Web Store assets

This folder keeps the editable materials for a future Chrome Web Store beta submission.

Use these files as source copy for the Developer Dashboard:

- `listing-en.md`: English Store Listing draft.
- `listing-pt-br.md`: Brazilian Portuguese Store Listing draft.
- `privacy-policy.md`: privacy policy draft.
- `test-instructions.md`: reviewer and local QA instructions.
- `release-notes.md`: beta release notes.
- `checklist.md`: publication checklist.
- `screenshots/`: screenshot guidance.
- `promo/`: promotional image guidance.
- `icons/`: icon guidance.

Before publishing, review all text against the current extension behavior, prepare real screenshots, and publish the beta as Private or Unlisted first.
