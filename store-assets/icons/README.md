# Icon Guidance

Use the `Polishing Pen Icon` concept for extension icons:

- Pen or pencil for editing.
- Small spark to communicate refinement.
- Do not use initials-based logos, badges, or avatars.
- No robot mascot.
- Corporate and discreet style.
- Confirm `icons/icon-128.png` exists and is referenced by `manifest.json`.

The Store Listing icon should match the packaged extension icon.
