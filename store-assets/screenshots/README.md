# Screenshot Guidance

Prepare real screenshots from the built extension. Do not use mockups that misrepresent the product.

Suggested screenshots:

- Extension popup.
- Refinement panel showing the text review step.
- Provider settings page.
- Sensitive-data warning.
- Example use in a real text field.

Screenshots should show the real Message Refiner experience, avoid third-party trademarks where possible, and avoid exposing personal, customer, or confidential data.
