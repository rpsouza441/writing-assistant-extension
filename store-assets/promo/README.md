# Promotional Image Guidance

Prepare a small promo tile for the Chrome Web Store:

- Size: 440x280.
- Use the Message Refiner visual identity.
- Use the polishing pen concept.
- Avoid excessive text.
- Avoid third-party brands and UI clones.
- Do not use initials-based logos, badges, or avatars.
- Keep the visual clean, professional, and suitable for a corporate writing tool.
